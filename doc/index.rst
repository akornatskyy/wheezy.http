wheezy.http
===========

Introduction
------------

:ref:`wheezy.http` is a `python`_ package written in pure Python code.
It is a lightweight http library for things like request,
response, headers, cookies and many others. It a wrapper around the `WSGI`_
request environment.

It is optimized for performance, well tested and documented.

Resources:

* `source code`_, `examples`_ and `issues`_ tracker are available
  on `github`_
* `documentation`_
